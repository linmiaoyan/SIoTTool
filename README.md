# 注意事项
0.2版本（基本功能+界面美化）：
可以在win10 部分win7运行
不能32位系统win7运行
报错1：“无法定位程序输入点ucrtbase.abort于动态链接库api-ms-win-crt-runtime-l1-1-0.dll上”
解决：进入微软官网进行下载运行时所需VC++组件，地址：https://www.microsoft.com/zh-cn/download/details.aspx?id=48145
